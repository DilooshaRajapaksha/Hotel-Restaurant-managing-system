# 🏨 Golden Stars Hotel & Restaurant Management System

A full-stack web application for managing hotel rooms, restaurant orders, and deliveries — built with React and Spring Boot.

🛠️ Tech Stack
 Frontend  - React, React Router, Axios 
 Backend - Spring Boot (Java) 
 Database - MySQL 
 Styling - CSS3, DM Sans, Playfair Display 
 API Testing - Postman 

We use mainly Agile method and manage it through the JIRA as our platform and we have 4 sprints each has 2 week. Out team manage a sprint backlog and sprint plan under the supervise of Project manager and our lectures. We have several roles and each role has responsibility in the project.

Project Manager - Manage the JIRA backlog and Sprint Backlog
		- Assign works to sprint backlog
		- Manage the clients works and others

Quality Assurance - Unit testing
		  - User acceptance criteria testing

Developers - Design the frontend 
	   - Implement the backend
	   - Connect the Backend frontend

### 🧑‍💼 Customer Section
- Register & Login
- Browse available hotel rooms
- View room details with photo gallery and 360° virtual tour
- Book rooms directly with check-in/check-out dates
- Browse restaurant menu
- Add items to cart and place food orders
- Track order status
- View booking history
- Manage profile

### 🚴 Delivery Section
- Delivery rider login
- View assigned delivery orders
- Update delivery status
- Track active deliveries

### 🔐 Admin Section
- Admin dashboard overview
- Manage rooms (Add / Edit / Delete)
- Manage room images including 360° panorama
- Manage restaurant menu items
- View and manage all bookings
- View and manage all orders
- Manage delivery assignments
- Manage users and roles

## ✨ Key Features

- 🖼️ **Image Gallery** — Multi-photo room viewer with thumbnail strip
- 🔮 **360° Virtual Tour** — Panoramic room viewer with interactive hotspots
- 🛒 **Cart System** — Add menu items and checkout
- 📅 **Room Booking** — Date picker with live price summary
- 🚴 **Delivery Tracking** — Real-time order status updates
- 🔒 **Protected Routes** — Role-based access (Customer / Delivery / Admin)
- 📱 **Responsive Design** — Works on desktop and mobile
- ✨ **Luxury UI** — Gold palette with smooth animations

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- Java 17+
- MySQL 8+

### Frontend Setup
```bash
cd FrontEnd
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
./mvnw spring-boot:run
```

### Environment
- Frontend runs on: `http://localhost:5173`
- Backend runs on: `http://localhost:8081`
